A Product Feed includes all the information related to products that are available on the marketplace for sale.

Multivendor Marketplacefor Shopify has now come up with a new feature app called Product Feed.

It enables the merchants to add products in bulk from different eCommerce platforms into Shopify via XML file upload.

Using this feature app, you can add feed by uploading an XML file.

This file is having products along with their information and details such as Product ID, Product Name, Product Price, its quantity and so on.

Once you enable this feature app installed on your marketplace, $10/month gets added to your current Multivendor Plan.

Feature

  • Admin can add Product Feed by uploading XML product file and assign it to any of his registered sellers.
  • This XML file is having all the data related to the products that you are selling.
  • All the product entities added in your XML file can be mapped with the pre-defined product entities available on your Shopify store.
  • Configure all related settings from his/her panel.
  • You can set a Price Rule on every feed that you add to your Marketplace.
  • Sellers registered on your marketplace can also add Feeds by adding Feed URL.

Limitations

There are certain limitations of this feature app listed below which you need to know.

  • Only XML file is supported while adding products in bulk using this feature app.
  • You cannot update the Feed Products.
  • You can't add variant wise Feed Products.
  • Multiple images can't be uploaded for a single Feed Product.

Installation

To make this app work on your marketplace, first of all, you need to enable this feature from the "Featured Apps" section of the Multivendor app.

Once you visit the Feature Apps section you will get this feature app. You just need to click the Enable button to activate this feature on your marketplace.

Once you click on the enable button you need to agree with the payment condition. Once you enable this feature $10 per month gets added to your current Multivendor plan.

By clicking the agree button you can approve the charge for this feature and get this app installed. In this way, you can get this app installed on your Marketplace.

Product Feed Configuration

Once you enable this app, you first need to configure the feed related settings from the Configuration section of the Multivendor app.

Visiting this Product Feed section you need to click on the "Add Product Feed" button to add related details of the product Feed.

Add Product Feed

There, you need to mention the Feed Name, set a Price Rule for this feed and mention the Start and End node of the feed.

Feed Name:- Enter the name of the product feed.

Price Rule:- Set the price rule for seller's feed products, Increase (increase the original price with a certain amount),

Decrease (decrease the original product price with a certain amount) and Neutral (keep the product price as it is).

Enter the Start & End Node. Please refer to the screenshot for reference:-

Here:-
Start Node:- Product
End Node:- Product

While setting up the Price Rule of the feed you can increase, decrease the Feed Product price as well as set it to Neutral.

Change in the Product Feed Price can be set in percentage or a fixed amount.

Once you are done with all these settings, just save the details and proceed to upload an XML file. Once uploaded, you need to validate the XML file.

By clicking this Save and Upload XMLyou will be reaching a page where you can upload a sample XML file that contains all the necessary information about a single product.

Once it gets uploaded you can validate the file and proceed to map the entities.

Once you validate the XML file, you can now proceed to map all the product entities available inthe Product Feed XML file with the pre-defined Shopify store product entities.

Click on the Save button to add this Product Feed.

Once you add this feed, you can view this in the Add Feed section of the app. Also you can edit the Product feed details as well as delete the Feed from Action menu.

In this way admin can add Feeds on his/her end.

Adding Feed Products

Now, You can add feed Products and assign it to your registered sellers. For this, you need to enter the following details:-

Seller Email:- Enter the seller's email ID for which you want to upload the product feed via the XML file.

Feed Name:- Enter the name of the feed.

Feed URL:- You need to mention the resource URL where the seller has added/uploaded/stored the XML file.

Now, once you save the details, you will get the options to map collection in your feed to the available collection.

You can view the list of Feed products in the Product Listing Section.

Admin can edit the feed product added, he can approve the product and also deny the product by mentioning the reason.

He can deny and delete the feed product at the same time also.

In case you want to deny the product, you can simply add a reason & unapprove the same.

Seller End

Once you enable the Product Feed feature app on your marketplace, your registered sellers can also add Feed Products from the Products section of the seller panel.

Here in this section sellers can add Products in bulk using XML file by adding Feed Products. Seller just needs to enter the Feed Name and the Feed URL to add Feed Products.

Feed URL:- The URL where the XML file of the products is uploaded.

The seller can view the list of added fields in the Product Listing section of the app.

Once the seller adds the Feed Product, he/she can view the list of Products in the Product Listing section of the Seller Panel and can anytime edit or delete the Product from the action menu.

This way you and your sellers can add Product Feeds using this feature app.
