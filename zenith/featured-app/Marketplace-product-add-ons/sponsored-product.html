Now promote your seller products on your Shopify store using the Sponsored Product feature app in the Multivendor Marketplace App

Purpose

This feature app allows the Admin to create plans for the Sellers so that they can promote their product on the Admin Shopify store.

Now you as an admin can promote your seller products on the Home page, Search, Product, and Collection page.

Admin can easily create different packages for each of these sections and sellers can buy and use these accordingly.

Update: Now admin can also allow sellers to set product-specific daily budgets and choose whether it would be Global or page-wise.

This flow is applicable when Stripe Usage Charge is selected as the payment method.

Note- Admin can set Minimum Threshold Budget restricting sellers from adding daily budget less than the threshold budget.

You will find more details about it in this blog below.

Now let's understand how you can configure and use this app for your Marketplace.

Installation and Workflow

To enable the feature app, visit the featured apps section of the Multivendor Marketplace for Shopify.

Now Activate the "Sponsored Product App" from the feature app section. After that, accept the charges and you are ready to use the app.

Once you enable the app, Two new submenus one under the Configuration "Sponsorship Configuration" And the second "Sponsored".

For the first you need to go to the Admin Panel >> Configuration >> Sponsorship Configuration

Here on this page, you will find a button to "Allow your seller to enable sponsorship on product".

So by enabling this, you can allow your seller to purchase the packages that the admin has created for their products.

Note: Please don't enable it before creating any Sponsorship package or cross-checking all details to avoid any sort of conflict between Admin & Seller

Workflow when using Stripe Usage Charge as the payment method

The workflow for the sponsorship feature when using stripe usage charge will be as follows:

Admin Panel:

  • Admin has to choose Stripe Usage Charge as the payment method.
  • Then he has to enter the Stripe Publish Key and Stripe Secret Key.
  • Then he has to choose the Billing Cycle, whether it would be weekly or monthly.
  • He can choose on which day of the week the weekly/monthly invoice should be generated
  • Admin can choose whether the Minimum Threshold budget should be Global or Page wise.

Admin can also allow sellers to set product-specific daily budgets and choose whether it would be Global or page-wise.

If the admin selects page-wise, he can navigate to the Sponsored list and set the page-wise budget.

If the admin sets it to Global, he can set the threshold budget there only.

Seller Panel:

In the Seller Panel, the seller has to add the billing and card details under Sponsor Payment Details.

After updating payment details, the seller can select a product in the Product Listing and set a sponsorship budget to enable sponsorship for that specific product.

The budget set by the seller would be treated as a daily spent budget, and it would be product-specific.

For example, if a seller sets a budget of 15 USD for a product, then 15 USD will be charged per day, and the bill will be generated weekly on Fridays.

So, the total amount generated will be 15 * (number of days). This would work as a recurring charge, and the sponsorship will continue indefinitely unless the seller manually stops it.

For sellers to download their receipts/invoices, they can go to the "Sponsor Order" section in the Seller dashboard. From there, they can view the details on the "Sponsor Order Description" page.
