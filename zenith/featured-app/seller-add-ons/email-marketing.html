Introduction

If you are planning to send bulk emails to sellers manually, and want to create an email campaign for your sellers then this marketplace featured app is for you.

Using this email marketing featured app you can connect klaviyo account with your multivendor marketplace app and your seller profiles will transfer to your klaviyo account also.

Installation

To install this featured app, go to the marketplace admin panel >> hover on the three dots >> Feature App section >>

Scroll down and search for "Email Marketing" featured app.

click the "Enable" button and Approve the charge & get this featured app to your marketplace.

As soon as you approve the app charges then in the admin panel, an "email marketing configuration" menu will appears under the configuration dropdown.

Click on the email marketing configuration.

Now, you will see an option to enable "KLAVIYO for email marketing", enter the KLAVIYO PRIVATE API KEY NAME and KLAVIYO PRIVATE API KEY.

Enable the "KLAVIYO for email marketing" and flow the below steps to get KLAVIYO PRIVATE API KEY NAME and KLAVIYO PRIVATE API KEY.

How to get KLAVIYO API KEY NAME and KLAVIYO PRIVATE API KEY

Step 1:- When you enter the Email marketing configuration screen in the admin panel then click on "By klaviyo" which will redirect to the klaviyo login page and login to your klaviyo account.

Step 2:- After you have a login, you will enter the Klaviyo dashboard. Now, click on Account plans >> >> setting >> click on API keys >> and click on create private API.

Step 3:- Now, first select the "full access key" then write the name of your private key API and click "CREATE" button.

Finally, you will get the klaviyo private API key.

Even you navigate through the Klaviyo dashboard Account plans >> >> setting >> click on API keys
then you can see your KLAVIYO PRIVATE API KEY NAME and KLAVIYO PRIVATE API KEY.

Now, you can enter this KLAVIYO PRIVATE API KEY NAME and KLAVIYO PRIVATE API KEY in th admin panel >> configuration >> email marketing configuraton and Click "SAVE " button.

Now, whenever the admin will create seller account or seller will sign up and create their account then automatically the seller profile will be created in your klaviyo account.

And when the seller profile is edited in the multivendor app then the same will be updated in your klaviyo account but vice versa will not work.
