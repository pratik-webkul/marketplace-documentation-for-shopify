In today’s e-commerce world, **print-on-demand products** are gaining significant popularity as customers increasingly seek personalized and unique items.

The ability to create and sell custom-designed products helps vendors meet customer needs and stand out in the market.

The [Webkul Multivendor Marketplace App](https://store.webkul.com/Shopify-Multi-Vendor-Marketplace.html)'s integration with **Print-On-Demand Products: Dynamic Mockups** makes selling personalized products more efficient and seamless than ever.

[Dynamic mockups](https://dynamicmockups.com/) allow sellers to add custom designs to products like T-shirts, mugs, or phone cases. These mockups give a realistic preview of how the product will look with the seller’s design.

By integrating this with the Webkul Multivendor Marketplace, sellers can easily manage and showcase their custom products, providing a smooth experience for customers.

 Features
---------------
**Editing**: You can easily modify images by adjusting colors, selecting print areas, and transforming perspectives.

**Automated Editing**: All changes are automatically applied, allowing you to view the final result instantly.

**Real-Time Changes**: See image adjustments in real-time as you make them.

 Getting Started with the Dynamic Mockup Print-on-Demand
---------------------------
To enable the Print on Demand - Dynamic Mockup feature, go to the **Multivendor Marketplace App Admin Panel**.

Hover over the three dots (...), select **Featured Apps**, then search for **Print on Demand - Dynamic Mockups** and click **Enable**.

![Print-On-Demand Products:](https://lh7-rt.googleusercontent.com/docsz/AD_4nXe_6yNRfVbbr-RivDD5i4nDuMjxzoswLwHqr456ATFzgzDlEhCAklN44P9TzQhsRuFUhF2m6CpuY6QM2QkAUOTkovft3cMC8XKR7aTUj_JKU6NULE7k7bSb-1ivzD55NswRkEi7KQ?key=FAR3e2gzSix_B3TCfXrjQg)

Once you enable this featured app, you need to approve the charges for the app i.e. $10 per month.

 Admin end Configuration
------------------
Once the charges are approved, the admin must configure the settings by navigating to **Multivendor Marketplace Admin Panel > Configuration > Print-on-Demand - Dynamic Mockups Configuration**.

In this section, the admin will add the API keys for both the admin and the seller to enable the Dynamic Mockup dashboard.

![Print-on-demand: Dynamic Mockups](https://lh7-rt.googleusercontent.com/docsz/AD_4nXf-YTtH7-VKmysVm-h1QYAdWXuw3Gew4AcdKpyqQnJ1Gex5Ilip50yrM3uZYMUdxnVFeLAWt_zR2l4Xvzs-_B-uzmqW1FzYbG5UlDMsrtm508Pt36reDtCsqrC1weu7FZQy8gYP?key=FAR3e2gzSix_B3TCfXrjQg)

To obtain the API keys, the admin can click on **"By Dynamic Mockups"** to create an account on the Dynamic Mockup platform and gain access to the dashboard.

![Print-on-demand: Dynamic Mockups](https://lh7-rt.googleusercontent.com/docsz/AD_4nXeKmrfqbm-us3yJZ8ED9S6iHegH8rNs_Y3MY32Hdcop2WeNT-_sLWw5g-YH2OZvs1k1Jmsdd5XfVMF9bRCr86tU2ZS5ViAm-XW9IcOTpCLltK8ORymYJvi6_QGpuHUgEFRn66AKPg?key=FAR3e2gzSix_B3TCfXrjQg)

 **For Admin Embeddable Mockup Editor Website Key**
-------------------------
Once you have created or logged into your account and filled in all the basic details, click on **'Embed Mockup Editor'** and then select **'Create New Website Key'**.

![Print-on-demand: Dynamic Mockups](https://lh7-rt.googleusercontent.com/docsz/AD_4nXeEVlSM21YDrBFpTYXZCotzO82lQgUv7b5VjkgA-y-26k6nm9oZIPwSEHgKPHHI4TDCZAtwdVhfZpdAgtPUjUmPIYAvQ92_mhpl-PMcJJCSBN7acdpRqnpxt1Vc4frVXDbvU7tAtg?key=FAR3e2gzSix_B3TCfXrjQg)

Clicking on 'Create New Website Key' will prompt you to enter your website name and the whitelisted domain ([https://sp-seller.webkul.com/](https://sp-seller.webkul.com/)). Then, click 'Create' to generate your API keys.

![Print-on-demand: Dynamic Mockups](https://lh7-rt.googleusercontent.com/docsz/AD_4nXea3Ixd5cNB8mib-8EvMOcYWFiTjBAkdF9zCdNmIhXfS_FiD42gVoF5ZjK_MY0-DVlVImfAyxxY73-p9pVo5EPtK1DU-BHFvXhGbkxiMoE_pUZjH1nnLGFkgTspFeWEDTZMEnwx?key=FAR3e2gzSix_B3TCfXrjQg)

 **For Seller Embeddable Mockup Editor Website Key**
---------------------------------------
To generate the **Seller API Key**, follow the same process as for the Admin. Click on **'Create New Website Key'**, enter your website name, and proceed to the whitelisted domain section.

In the whitelisted domain section, enter your **seller URL** from the **Multivendor Marketplace App**. Finally, click **'Create'** to generate your API keys.

![Print-on-demand: Dynamic Mockups](https://lh7-rt.googleusercontent.com/docsz/AD_4nXe25_VGKglJrtu-P7CuxCgXww_4_cQT8uiDdkrGQZq7M6LMfAVjl4eSehtyYUJY51bqqeBxF0WxbWOqrYvrW0o0sz74VbMVlwyWuYwhm4iy-e2gC6TAfz_UBy6N-d2GM7PHaF2q?key=FAR3e2gzSix_B3TCfXrjQg)

After generating the API keys for both Admin and Seller, copy the website key and enter it into the Configuration section.

![Print-on-demand: Dynamic Mockups](https://lh7-rt.googleusercontent.com/docsz/AD_4nXfWF-vIPBtLHbaYxVRky1JpBYgioE8bk0NuNc2MqhM4vTS9XY88pIr3xZMErxsqWgItwxUvCTOdCQefbn2M_l45qlZW3Rz-sqcN7ncSiK2Q-MaVSMLsmw2vGCnoXToNk1KCG8VY?key=FAR3e2gzSix_B3TCfXrjQg)

Once you (Admin) register yourself on the Dynamic Mockups platform, you can add mockups like T-shirts, mugs, pillows, etc., where you want to print your design.

![Print-on-demand: Dynamic Mockups](https://lh7-rt.googleusercontent.com/docsz/AD_4nXfisVD3RDkWKvluYGo7zj-27iamW07n3TOHqJJii44v78UksKydibUQ_CXWH2zGoKs8jXAC1fd1r82nbXJE0U7wKZf4BodLR1ga1XqpxPE7Si8aR2AldQmZcdMHXboAHfAqNtoJ?key=FAR3e2gzSix_B3TCfXrjQg)

 Seller Panel
-------------------------
Once the Admin sets the mockups, the Seller can use the **Dynamic Mockups** feature while adding their product.

The Seller needs to check the box **'TO ADD PRINT-ON-DEMAND PRODUCT - DYNAMIC MOCKUPS'** on the product form page.

When this checkbox is selected, the option **'Image add by POD'** becomes available on the product form page.

![Print-on-demand: ](https://lh7-rt.googleusercontent.com/docsz/AD_4nXd_cZvDe9CPLRZqt9MaGclhRn8bTxImKHUwVReHU4fzGVgpY2x-9MDEPZjaUUS1vX4jm6G8NEFL_eEMIW36H0i8Dkr0YhT7Oc7kgZnZCj4OUFhT0lpioP9QJ9kybxDHrnfLwkHKcw?key=FAR3e2gzSix_B3TCfXrjQg)

When the seller clicks on **'Add Images by POD'**, the **Print-On-Demand Products: Dynamic Mockups** screen appears.

![Print-on-demand: Dynamic Mockups](https://lh7-rt.googleusercontent.com/docsz/AD_4nXdXguOm7Q_hbqJ7OYFOfs_v2RsgQyU4jlJ65fNIEuGByulfHYd71VZY2TMwQjWIK1-EZ-6cMPQhqeh3e0SmG7cicGuXHCnatiSzrcKlNP2f3ZUJ2qrBcmY3HBGXDLjd-C199LtR?key=FAR3e2gzSix_B3TCfXrjQg)

From there, the seller can select a product that the Admin has already added, upload their design, and adjust its placement on the product.

They can preview how the design will look in real-time and also change the product's color as needed.

**Note:** A maximum of 10 images can be added.

![Print-on-demand: ](https://lh7-rt.googleusercontent.com/docsz/AD_4nXcIYVW5rASRnc_qtn3twUyXS6HYOuNQcHX2HLtHm929acmW7GOO7OTivgPV3NZ2E6liwfYmxmWblCjqsrpLrUZkVRWZ9iCi5nApa0GrgHKqA7k2vtbn4SpK4I4T7EwNvrYf0pWblQ?key=FAR3e2gzSix_B3TCfXrjQg)

The seller can click on the **plus (+) icon** to add their design for printing on the product and use the **color option** to change the product's color.

![Print-on-demand: Dynamic Mockups](https://lh7-rt.googleusercontent.com/docsz/AD_4nXenZW4aKlfOyN-E5DkMuyO4ZL89PkQ2lq8962UfAftvbQNbXj2SbbnHC6HnwJTnSbQ9Ny0yus0OHs3yftPPeTGeY_yfbtHquU_jJmHYmIfSe5bTtNQ7GT1uHLh3F_Db2IVeCK8Ntw?key=FAR3e2gzSix_B3TCfXrjQg)

Adjust the design on the product to your desired position, then click on the **'Export Mockups'** tab to export the design.

Afterward, the seller can fill in all the product details on the product form page and click **'Save'** to complete the process.

**Note:** Once a product is marked as **POD**, it cannot be unchecked again.

When the seller adds the product, it will appear in the **Product Listing** section of the seller panel.

![Print-on-demand: ](https://lh7-rt.googleusercontent.com/docsz/AD_4nXcLinzOSA9FjK6jweNsAOBWgRtvpqgOB89gHpyMheJe2Z5apjr4j8D0y2CLgwYqeYDAI3JsbixvHCVXk6Q_0TfGsdTnQ84oupHgRHDWz4LE3ILhEuvNFlqKTrx-vQx5ReSmfP7z?key=FAR3e2gzSix_B3TCfXrjQg)
